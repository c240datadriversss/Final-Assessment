## üîÑ Customization Guide

### **To Add Ingredients:**
1. Edit `js/chatbot.js`
2. Find `this.substitutions = {`
3. Add new ingredient object with alternatives
4. Save and test

### **To Change Colors:**
1. Edit `css/style.css`
2. Find `:root {` section
3. Update color variables
4. Changes apply site-wide

### **To Update Content:**
1. Open corresponding `.html` file
2. Edit text in HTML
3. Styling automatically applies

### **To Deploy:**
1. `git add .`
2. `git commit -m "message"`
3. `git push origin main`
4. Enable GitHub Pages
5. Done!
