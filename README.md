### Usage

1. **Using the Chatbot:**
   - Click the chat icon (ğŸ’¬) in the bottom-right corner to open the chatbot
   - Type your ingredient substitution question (e.g., "What can I substitute for butter?")
   - Receive instant recommendations with ratios and tips
   - The chatbot input will auto-expand as you type longer messages

   - Navigate to the "Guide" page for a comprehensive list of substitutions
